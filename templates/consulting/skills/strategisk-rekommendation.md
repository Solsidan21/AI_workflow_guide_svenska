# Skill: Strategisk rekommendation

## Beskrivning

Denna skill aktiveras när användaren behöver formulera en strategisk rekommendation — oavsett om det gäller marknadsinträde, organisationsförändring, förvärv, investeringsbeslut eller annan strategisk frågeställning. Rekommendationen ska följa pyramidprincipen och vara direkt handlingsbar.

**Triggers:** "strategisk rekommendation", "strategiförslag", "vad bör vi göra", "rekommendera en väg framåt", "strategisk rådgivning", "handlingsplan"

---

## Steg-för-steg-instruktioner

### Steg 1: Förstå frågeställningen

Klargör med användaren:
- Vad är den centrala frågeställningen? (Formulera som en ja/nej- eller val-fråga)
- Vem är beslutsfattaren?
- Vilka begränsningar finns? (tid, budget, organisation, regulatoriska)
- Vilken data finns tillgänglig?
- Vilka alternativ har redan övervägts?

### Steg 2: Strukturera analysen med SCR

Använd Situation — Complication — Resolution som övergripande struktur:

**Situation:**
- Nuläge — var befinner sig organisationen idag?
- Bakgrund — vad har lett fram till denna frågeställning?
- Styrkor att bygga vidare på

**Complication:**
- Vad har förändrats eller utmanar nuläget?
- Varför fungerar inte status quo?
- Vad händer om inget görs? (cost of inaction)
- Vilken tidsfrist finns?

**Resolution:**
- Rekommenderad väg framåt
- Varför just detta alternativ?
- Förväntad effekt (kvantifierad)
- Nyckelrisker och hur de hanteras

### Steg 3: Utvärdera alternativ

Identifiera och utvärdera 2–4 realistiska strategiska alternativ:

| Kriterium | Alternativ 1 | Alternativ 2 | Alternativ 3 |
|-----------|-------------|-------------|-------------|
| Beskrivning | | | |
| Intäktspotential | | | |
| Investeringsbehov | | | |
| Tidsram till effekt | | | |
| Risknivå | | | |
| Organisatorisk komplexitet | | | |
| Strategisk passform | | | |
| **Helhetsbedömning** | | | |

Analysera varje alternativ med:
- **Fördelar** — vad talar för detta alternativ?
- **Nackdelar** — vad talar emot?
- **Förutsättningar** — vad krävs för att det ska fungera?
- **Risker** — vad kan gå fel?

### Steg 4: Riskbedömning

Identifiera och bedöm risker kopplade till den rekommenderade strategin:

| Risk | Sannolikhet | Påverkan | Mitigering |
|------|-------------|----------|-----------|
| | Hög/Medel/Låg | Hög/Medel/Låg | |

Kategorisera:
- **Strategiska risker:** Marknadsförändringar, konkurrentreaktioner, teknologiskiften
- **Exekveringsrisker:** Organisatorisk förmåga, resursbrist, tidplan
- **Finansiella risker:** Investeringsutfall, kassaflödespåverkan, finansiering
- **Externa risker:** Konjunktur, regulatorik, geopolitik

### Steg 5: Implementeringsplan

Skapa en konkret handlingsplan:

**Fas 1: Snabba vinster (0–3 månader)**
- Initiativ som kan startas omedelbart
- Kräver minimala investeringar
- Bygger momentum och trovärdighet

**Fas 2: Strukturella förändringar (3–12 månader)**
- Större initiativ som kräver planering
- Organisations- och processförändringar
- Investeringsbeslut

**Fas 3: Långsiktig transformation (12–36 månader)**
- Strategiska positionsförändringar
- Nya förmågor och kapaciteter
- Kulturförändring

Per initiativ, specificera:
- Ansvarig
- Resursbehov
- Milstolpar
- Framgångskriterier (KPI:er)
- Beroenden

### Steg 6: Formulera rekommendationen

Följ pyramidprincipen strikt:

1. **Slutsats** (en mening): Vad rekommenderar vi?
2. **Stödjande argument** (3–4 punkter): Varför rekommenderar vi detta?
3. **Data och analys**: Vilken evidens stödjer argumenten?

Varje stödjande argument ska i sin tur följa samma struktur:
- Påstående
- Stödjande data
- Implikation ("so what?")

## Exempelprompt

> "Vi rådger en svensk industrigrupp med 2 MDSEK i omsättning som överväger att expandera till den tyska marknaden. De har tre alternativ: organisk etablering, förvärv av en lokal aktör, eller partnerskap med en tysk distributör. Kan du hjälpa oss att strukturera en strategisk rekommendation? Vi har marknadsstorlek, konkurrentdata och en preliminär finansiell analys."
