# Exempeldata och exempelanalyser

Denna mapp är till för exempeldata och exempelanalyser. Lägg till anonymiserade exempel här som Claude kan använda som referens för att förstå vilken kvalitet och format ni förväntar er.

## Vad kan läggas här

- Anonymiserade kassaflödesanalyser
- Exempelutdrag från investment memos (anonymiserade)
- Fiktiva dataset för att testa Claudes analysramverk
- Exempel på styrelserapporter (med fiktiva siffror)

## Viktigt om datasekretess

**Lägg ALDRIG konfidentiell data i denna mapp utan att först anonymisera den.**

Att anonymisera innebär:
- Byt ut bolagsnamn, personnamn och kundnamn
- Justera eller skala om finansiella siffror
- Ta bort geografisk information som kan identifiera bolaget
- Ta bort datum som kan kopplas till specifika transaktioner

Tänk på att denna mapp kan delas med kollegor eller versionshanteras i git. Behandla allt innehåll som om det är publikt tillgängligt.
