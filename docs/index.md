# Claude Instruktionsbok — Från Chat till Code

**En praktisk guide för kunskapsarbetare som vill använda AI effektivt**

---

Den här guiden tar dig från att använda Claude som en enkel chattbot till att utnyttja hela AI-ekosystemet — inklusive Claude Code, automatisering och avancerade arbetsflöden.

Guiden är skriven för **kunskapsarbetare** som inte är utvecklare men som vill få mer gjort med AI:

- **PE-rådgivare och revisorer** som arbetar med analys, rapporter och klientkommunikation
- **Managementkonsulter** som bygger presentationer, strategidokument och beslutsunderlag
- **Marknadsprofessionella** som skapar innehåll, kampanjtexter och dataanalys

Du behöver inte kunna programmera. Du behöver inte ha öppnat en terminal förut. Allt förklaras steg för steg, på svenska, med analogier från den värld du redan känner.

## Kom igång

Du kan läsa guiden på två sätt:

1. **Från början till slut.** Börja med kapitel 00 och arbeta dig framåt. Varje kapitel bygger på det föregående och tar dig gradvis längre in i AI-världen.

2. **Hoppa direkt till det du behöver.** Om du redan använder Claude Chat dagligen kanske du vill börja vid kapitel 03 (Excel och PowerPoint) eller 04 (Claude Code). Varje kapitel fungerar rimligt bra på egen hand.

Oavsett vilken väg du väljer, rekommenderar vi att börja med **kapitel 00** — det tar bara några minuter och ger dig ordförrådet som resten av guiden bygger på.
