# Claude Instruktionsbok — Fran Chat till Code

**En praktisk guide for kunskapsarbetare som vill anvanda AI effektivt**

---

## Om den har guiden

Det har ar en svensk instruktionsbok som tar dig fran att anvanda Claude som en enkel chattbot till att utnyttja hela AI-ekosystemet — inklusive Claude Code, automatisering och avancerade arbetsfloden.

Guiden ar skriven for **kunskapsarbetare** som inte ar utvecklare men som vill fa mer gjort med AI. Tankt malgrupp:

- **PE-radgivare och revisorer** som arbetar med analys, rapporter och klientkommunikation
- **Managementkonsulter** som bygger presentationer, strategidokument och beslutsunderlag
- **Marknadsprofessionella** som skapar innehall, kampanjtexter och dataanalys

Du behover inte kunna programmera. Du behover inte ha oppnat en terminal forut. Allt forklaras steg for steg, pa svenska, med analogier fran den varld du redan kanner.

## Kom igang

Du kan lasa den har guiden pa tva satt:

1. **Fran borjan till slut.** Borja med kapitel 00 och arbeta dig framat. Varje kapitel bygger pa det foregaende och tar dig gradvis langre in i AI-varlden.

2. **Hoppa direkt till det du behover.** Om du redan anvander Claude Chat dagligen kanske du vill borja vid kapitel 03 (Promptteknik) eller 04 (Excel och PowerPoint). Varje kapitel fungerar rimligt bra pa egen hand.

Oavsett vilken vag du valjer, rekommenderar vi att borja med **kapitel 00** — det tar bara nagra minuter och ger dig ordforradet som resten av guiden bygger pa.
